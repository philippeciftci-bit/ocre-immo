Documentation Webhooks Ocre Immo

Intègre Ocre Immo à ton CRM, Zapier, Notion ou Google Sheets via webhooks. POST JSON signés HMAC-SHA256 envoyés en temps réel quand des events se produisent dans ton tenant.

Configuration

Dans Oi Agent : Réglages → Intégrations → + Nouveau webhook → entre l'URL de ton endpoint + sélectionne les events à recevoir → tu obtiens un secret one-shot à conserver.

Vérification de la signature

Recompose la signature côté serveur avec ton secret pour vérifier l'authenticité du POST. Compare avec X-Ocre-Signature en time-constant (utilise hmac.compare_digest ou équivalent).

Node.js

const crypto = require('crypto');
function verify(rawBody, signature, secret) {
  const expected = 'sha256=' + crypto.createHmac('sha256', secret).update(rawBody).digest('hex');
  return crypto.timingSafeEqual(Buffer.from(expected), Buffer.from(signature));
}

Python

import hmac, hashlib
def verify(raw_body: bytes, signature: str, secret: str) -> bool:
    expected = 'sha256=' + hmac.new(secret.encode(), raw_body, hashlib.sha256).hexdigest()
    return hmac.compare_digest(expected, signature)

PHP

function verify(string $rawBody, string $signature, string $secret): bool {
    $expected = 'sha256=' . hash_hmac('sha256', $rawBody, $secret);
    return hash_equals($expected, $signature);
}

Retry policy

  • Timeout : 5s par requête
  • Retries : exponentiel 1m → 5m → 15m → 1h → 6h → 24h max 6 essais
  • Auto-disable : après 5 échecs consécutifs, le webhook est désactivé automatiquement et l'agent reçoit une notif PWA push
  • Réactivation : depuis Réglages → Intégrations, tape "Réactiver" sur le webhook désactivé (le compteur de failures repart à 0)

Limitations

  • Payload max 256 KB par event
  • HTTPS uniquement (HTTP rejeté à la création)
  • Max 10 webhooks par tenant
  • Rate limit : 100 events/heure/webhook

FAQ

Comment tester un webhook avant production ?

Utilise webhook.site ou requestbin.com pour générer une URL temporaire qui affiche les payloads reçus.

Comment intégrer Zapier ?

Crée un Zap → trigger "Catch Webhook" → copie l'URL → utilise-la comme webhook URL dans Ocre.

Comment intégrer Notion ?

Via Zapier ou Make.com (intégration native Notion + webhooks input). Ou direct API Notion via un middleware custom (Cloudflare Workers, AWS Lambda).
