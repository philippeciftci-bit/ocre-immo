Export / Import ZIP

📦 Exporter un dossier en ZIP autonome

Génère un ZIP contenant data.json + photos + viewer.html standalone offline. À envoyer par email/AirDrop à un confrère externe ou un client. Le destinataire double-clique sur viewer.html pour voir le dossier sans connexion internet.

Nom/prénom/email/tél → "[Anonyme]"
Tous les montants → "Sur demande"
PDF + scans (peut alourdir le ZIP)

📥 Importer un dossier reçu (ZIP)

Si un confrère t'a envoyé un ZIP Ocre Immo, importe-le ici pour créer un nouveau dossier dans ton tenant. Photos + Section III financière copiées intégralement.

Le ZIP doit avoir été généré par Ocre Immo ≥ v18.32. Les anciens formats ne sont pas compatibles.
